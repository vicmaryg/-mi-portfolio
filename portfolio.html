Descripción del Proyecto

Desarrollo completo del sitio web oficial de la Fundación Cruxes de Luz, una organización que promueve la espiritualidad, solidaridad y desarrollo personal. El proyecto incluye:

  • Internacionalización: Sistema completo de traducciones con archivos JSON
  • Diseño Responsivo: Mobile-first con breakpoints optimizados
  • Sistema de Donaciones: Integración con MercadoPago para donaciones únicas y recurrentes
  • Gestión de Privacidad: Modal de consentimiento de cookies GDPR/LOPD
  • Optimización: Imágenes WebP, CSS modular, JavaScript optimizado
  • Accesibilidad: Navegación por teclado, contraste adecuado, textos alternativos
